# RN-GitHub

A Github client of React Native.

全面使用hooks重构的版本

## ✨ 技术栈

*  react 17
*  react native 0.68
*  react-navigation 6
*  react-native-vector-icons

## 🔨 快速开始

```bash
$ git clone https://github.com/YanYuanFE/RN-GitHub.git
$ cd RN-Github
$ yarn
$ cd ios && pod install // ios
$ react-native run-android / run-ios
```
